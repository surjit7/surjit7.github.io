AMT Concrete is a Production-Ready Kubernetes Hosting Platform that launches and runs your entire application stack on a single pre-configured virtual machine with no DevOps expertise required.

## What Is This Image?
- A pre-configured, production-grade virtual machine image
- Bundles Kubernetes and essential cloud-native tools
- Enables instant application hosting with minimal infrastructure or DevOps knowledge on supported VM sizes
- Preloaded with all required services for hosting

## What's Included
- **NGINX Ingress Controller** - Traffic management and load balancing
- **HashiCorp Vault** - Auto-init and unseal for secret management
- **Argo CD** - GitOps-style deployments
- **Redis** - In-memory data structure store
- **Kafka** - Distributed streaming platform
- **MongoDB** - NoSQL database
- **Kubernetes Dashboard** - Visual cluster management
- **MySQL** - Relational database
- **PostgreSQL** - Advanced relational database
- **Chroma DB** - Vector database for AI/ML
- **Keda** - Event-driven autoscaling
- **Velero** - Kubernetes backup and disaster recovery
- **Metrics Server** - Cluster-wide resource metrics
