The AMT Scaler extends your AMT Concrete environment into a fully scalable Kubernetes cluster with Azure Virtual Machine Scale Sets (VMSS). Designed for workloads that need elasticity, high availability, and enterprise-grade performance.

## What Is This Image?
- A pre-configured, scale-ready virtual machine image
- Works as the scalable companion to AMT Concrete
- Integrates seamlessly with Azure VMSS for automatic scaling
- Transforms a single-node cluster into a multi-node production-grade setup
- Preloaded with orchestration, monitoring, and data services to support scaling workloads

## Plug-and-Scale Infrastructure
- **Elastic Growth**: Scale worker nodes automatically with Azure VMSS
- **High Availability**: Achieve resilience by eliminating single points of failure
- **Seamless Integration**: Built on AMT Concrete base — migrate instantly to scale-out mode
- **Cost Efficient**: Scale resources up or down based on demand
- **Cloud-Native Ready**: Optimized for production apps, microservices, and heavy data workloads

## Why Choose AMT Scaler?
- **From Host to Scale**: Expand from a single-node to a resilient, multi-node cluster
- **Enterprise-Grade**: Handle mission-critical workloads with confidence
- **Zero Lock-In**: Runs on Azure VMSS, under your control, without dependency on managed Kubernetes services
- **Future-Ready**: Supports CI/CD pipelines, GitOps, and high-volume applications

## Key Features
- Multi-node Kubernetes Cluster powered by Azure VMSS
- Elastic scaling of worker nodes
- Integrated with Argo CD, Vault, and Kubernetes Dashboard
- Supports Kafka, Redis, and MongoDB for scalable workloads
- Calico CNI for secure, scalable networking
- NGINX Ingress Controller for traffic management and load balancing
- Monitoring-ready: Extendable with Prometheus, Grafana, and logging stacks
