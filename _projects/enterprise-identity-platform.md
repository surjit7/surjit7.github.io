The Enterprise Identity Platform is a production-ready OAuth2 & OpenID Connect multi-tenant authentication platform built with Django. It provides a scalable, secure solution for managing users, clients, and permissions across multiple tenants.

## Key Features
- **OAuth2 & OpenID Connect** - Full compliance with industry standards
- **Multi-tenant Architecture** - Support for independent tenant isolation
- **JWT Authentication** - Secure token-based authentication
- **RBAC** - Role-Based Access Control for fine-grained permissions
- **Azure AD Integration** - Seamless integration with Microsoft Entra ID
- **Scalable & Secure** - Designed for enterprise-scale deployments

## Technology Stack
- Python, Django, Django REST Framework
- OAuth2, OpenID Connect, JWT
- PostgreSQL, Redis
- Docker, Nginx, Gunicorn
