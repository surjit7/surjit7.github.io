Enterprise Identity Platform

Production-grade Identity Provider (IdP) implementing OpenID Connect (OIDC) and OAuth 2.0 with full multi-tenant isolation, RBAC, and JIT 2FA. Serves as the central authentication foundation that other services depend on for secure authentication and authorization.

Key Engineering Challenges Solved

  • Multi-tenant architecture with complete tenant isolation across independent identity domains
  • OAuth2 & OpenID Connect compliance (RFC 6749, RFC 8252) with three grant types: Authorization Code (Web), Device Code (Headless), Client Credentials (Server-to-Server)
  • Just-In-Time (JIT) 2FA enrollment with TOTP QR code setup and recovery codes
  • Session binding via _sso_sid claim in access tokens for revocation tracking and token correlation

My Role & Contribution

Designed and implemented the core OAuth2/OIDC authentication flows, multi-tenancy engine, RBAC permission resolution, and JIT 2FA system for enterprise-scale deployment.
